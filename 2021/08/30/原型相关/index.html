原型相关

发表于 2021-08-30 分类于 前端知识

原型（new）

proto 隐式原型
foo.proto 可以找到实例中的函数
Foo.prototype === foo.proto
每个对象都有一个隐式原型，都会指向显式原型

new 实例对象:通过实例的方式构造出来的对象。
var foo = new Foo()
原型：构造函数通过new 的方式执行会改变this指向，并赋值给一个变量，就成了实例对象，这时候Foo就是foo实例的原型。
构造函数的prototype指向原型对象，原型对象有一个constructor属性指向构造函数，每个构造函数生成的实例对象都有一个proto属性，这个属性指向原型对象
Foo.prototype 原型本身
foo.proto 实例对象获取原型
Object.getPrototypeOf(foo) === Foo.prototype

原型链
当我们访问对象的属性方法时，会从自身的原型对象去找，如果找不到就往原型的原型去找属性方法，这样就形成了原型链
使用：用过实例对象.hasOwnProperty(’属性’) == true则本原型中有此属性，false则没有

foo.proto == new Foo( ) -> Foo.prototype -> Object.prototype
本实例对象 == new 原型实例对象->父类实例对象->Object

instanceof 查找这个原型对象的是否在原型链内

函数优先级中括号( )> . >new
1.this指向当前方法的默认obj对象
2.用prototype原型来给构造函数添加this.函数名
Person.prototype.show = function(){}

继承的全局this不能被事件（onclick）使用
function abc (){
Var _this = this;
this.oBtn[0].onclick = function(){
_this.tab(this)
}
}
abc.prototype.tab = function(thisBtn){
this.oBtn[0].style.display = “none”;
thisBtn.style.display=“block”;
}

ECMA6 class语法

constructor(name, sex, age){
this.name = name;
this.sex = sex;
this.age = age;
}
showSelf(){
alert(${this.name},${this.sex},${this.age})
}
static info = ‘eee’//静态属性

}
var p1 = new Person(“blue”，”男”，18)；
p1.showSelf()

静态属性
1、不能被实例继承————-p1.info //报错
2、只能直接通过类调用——–Person.info
3、可以被子类继承————-Worker.info
4、可以从super对象上调用—-super.info

extends 继承
class Worker extends Person{
constructor(name,sex,age,job){
super(name,sex,age); //继承后必须优先调用super函数，引用父类的构造器内容
this.job = job;
}
showJob(){
alert(“我的工作是”+this.job)
}
}
var w1 = new Worker(“小米”,男，20，程序员)
w1.showSelf();
w1.showJob();
